## Get started

<Steps>
  <Step title="Sign in to the dashboard">
    Go to [app.objectionly.com](https://app.objectionly.com/) to create an
    account or sign in.
  </Step>
  <Step title="Connect an integration or request an API key">
    Choose an integration from the list above, or request an API key if you
    plan to use the API via the
    [API key request form](https://docs.google.com/forms/d/e/1FAIpQLSeWq6YmZSFyKS_wIO_enSdDiMygH22HUIjR2Atu3A0D4aGH1g/viewform).
  </Step>
  <Step title="Queue calls for processing">
    Send call transcripts or audio files and track their status in Objectionly.
  </Step>
</Steps>
