---
title: How Rep Access Works
description: Learn how Admins assign rep visibility to Members in Objectionly.
---

Admins can see every rep in the workspace. Members only see the rep data an Admin gives them access to.

## Three types of Member access

<CardGroup cols={3}>
  <Card title="All reps" icon="users">
    The Member can view calls and reports for every rep.
  </Card>
  <Card title="Specific reps" icon="user-check">
    The Member can view only the reps an Admin selects.
  </Card>
  <Card title="No reps assigned" icon="user-x">
    The Member can log in, but cannot see rep data yet.
  </Card>
</CardGroup>

## What rep access controls

When a Member has access to a rep, they can see:

- That rep's dashboard and reports
- Calls and transcripts for that rep
- Scorecards, objections, and coaching insights for that rep
- That rep in leaderboards

Rep access does not grant admin permissions. Members still cannot upload calls, edit calls, manage reps, change settings, or invite teammates.

<Tip>
  If a Member only has access to one rep, Objectionly takes them straight to
  that rep's dashboard when they log in.
</Tip>
