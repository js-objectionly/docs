# Call Processing API

Use the Objectionly Call Processing API to queue calls for processing with a
transcript or audio file.

<Note>
  Request your workspace API key via the
  [API key request form](https://docs.google.com/forms/d/e/1FAIpQLSeWq6YmZSFyKS_wIO_enSdDiMygH22HUIjR2Atu3A0D4aGH1g/viewform).
</Note>

## Base URL

[https://api.objectionly.com](https://api.objectionly.com)

## Authentication

<ParamField header="X-Workspace-API-Key" type="string" required>
  Include your workspace API key with every request.
</ParamField>
